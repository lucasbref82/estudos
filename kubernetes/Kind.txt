Cria um cluster:
kind create cluster

Criar um cluster com nome:
kind create cluster --name {NOME DO CLUSTER}

Criar um cluster com nome e com uma configuração de um manifesto.
kind create cluster --name {NOME DO CLUSTER} --config {CONFIGURACAO DO CLUSTER.yaml}

Listar todos os clusters:
kind get clusters

Deletar todos os clusters:
kind delete clusters

Delete um cluster pelo nome:
Kind delete cluster --name {NOME DO CLUSTER}
